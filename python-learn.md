## About This Repository

This repository contains structured lesson notes, code examples, and reference documentation following the CS50 Python course curriculum. Each lesson folder holds a dedicated `lesson##.md` file covering the concepts introduced in that session.

The goal is to build a reliable learning reference — written in a way that is approachable for beginners while remaining precise enough to be useful as ongoing documentation.
